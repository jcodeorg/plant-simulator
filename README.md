# 植物成長シミュレーター 技術仕様書 (サニーレタス版)

本ドキュメントは、サニーレタスの生理生態に基づいた「代謝ベース成長モデル」を採用したシミュレーションエンジンの仕様を規定する。

## 1. 概要
本シミュレーターは、単なる環境条件による成長計算ではなく、**「昼のエネルギー蓄積（光合成）」**と**「夜の組織構築（代謝・転流）」**を分離してシミュレートする。これにより、24時間点灯の弊害や、夜間の温度管理による徒長（とちょう）のリスクをリアルに再現する。

---

## 2. 状態変数 (State Variables)

| 変数名 | 定義 | 範囲 | 影響内容 |
| :--- | :--- | :--- | :--- |
| `growth` | 総成長量 | 0.0 ~ 1.0 | 1.0で収穫可能。葉の面積・実質的な重量を表す。 |
| `damage` | 健康被害度 | 0.0 ~ 1.0 | 1.0で枯死。環境ストレスの累積。 |
| `storageDLI` | 貯蔵エネルギー | 0.0 ~ 1.0 | 昼にチャージし、夜に消費するバッテリー。 |
| `waterLevel` | 水位差 (cm) | 負数 ~ 0.0 | 0.0が基準。-3.0cm以下で致命的ダメージ。 |
| `tipburn` | チップバーン度 | 0.0 ~ 1.0 | 高負荷・蒸散不良による葉先の枯れ。 |
| `etiolation` | 徒長/軟弱化度 | 0.0 ~ 1.0 | 光不足や夜温過剰による組織の脆弱化。 |

---

## 3. コア・ロジック

### 3.1 飽和水蒸気圧差 (VPD)
温度($T$)と湿度($H$)から算出。
- **計算式:** テテンの式による飽和水蒸気圧 $e_{sat}$ と現在水蒸気圧 $e_{air}$ の差。
- **最適域:** 0.8 〜 1.2 kPa。
- **影響:** 成長係数 $fVPD$、蒸散速度（水位減少）、およびチップバーン発生率。

### 3.2 昼間フェーズ（$L > 50\text{ lx}$）
- **蓄積:** 光応答 $fL$ と温度応答 $fT$ に基づき `storageDLI` を加算。
- **光不足判定:** 光量が $5000\text{ lx}$ 未満の場合、`isDayLightDeficit` フラグを真にする。
- **デンプン飽和:** `storageDLI > 0.7` で光合成抑制（0.8倍）および微量なチップバーン蓄積を開始。
- **即時成長:** 一部のエネルギーは昼間も `growth` に変換される。

### 3.3 夜間フェーズ（$L \le 50\text{ lx}$）
- **変換:** `storageDLI` を消費（毎時最大 0.05）し、`growth` または `etiolation` へ変換。
- **変換効率:** 夜間温度 $fT$ が最適に近いほど高い。
- **ルート分岐:**
    - **正常ルート:** `isDayLightDeficit` が偽の場合。エネルギーを健全な `growth` に変換。
    - **徒長ルート:** `isDayLightDeficit` が真の場合。エネルギーを `etiolation`（伸長）に優先配分し、`growth` 効率は 0.2 倍に低下。
- **軟弱化判定:** 光は足りていても夜温が $22^\circ\text{C}$ を超える場合、緩やかに `etiolation` が進行。

---

## 4. 成長ステージ・パラメータ

成長率（`growth`）に応じて特性が変化する。

| ステージ | 成長率範囲 | 特徴 | 徒長感度 |
| :--- | :--- | :--- | :--- |
| **発芽期** | 0.00 - 0.10 | 暗所を想定。徒長は発生しない。 | 0.4 |
| **幼苗期** | 0.10 - 0.30 | **最重要期間。** わずかな光不足で激しく徒長する。 | **1.5** |
| **生育期** | 0.30 - 0.70 | 旺盛に成長。チップバーンリスクが顕在化。 | 0.7 |
| **成熟期** | 0.70 - 1.00 | 低温（18℃）を推奨。苦味抑制と品質向上。 | 0.3 |

---

## 5. ダメージとストレスの定義

最終的なダメージ増加量 `netDamage` は以下の合算となる。

1. **環境ストレス (`dmgE`):** 高温（$> 26^\circ\text{C}$）および湿度異常（最適 60% からの乖離）。
2. **水位ストレス (`dmgW`):** 水切れ（$-3.0\text{cm}$ 以下）時に急増。
3. **チップバーン (`dmgT`):** 強光・高温・VPD異常（$< 0.4$ または $> 1.8$）の積。
4. **徒長/軟弱化 (`dmgEt`):** 代謝異常による組織劣化。**自然回復の対象外。**

---

## 6. 特筆事項
- **非可逆性:** 組織の徒長およびチップバーンは、環境が改善しても蓄積されたままであり、収穫時の品質スコアを下げる。
- **夜の必要性:** 24時間点灯を続けると `storageDLI` が飽和し、夜間の効率的な組織構築が行われないため、不健康な個体となる。
